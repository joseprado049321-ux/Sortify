# 🚀 Sortify // Organizador Inteligente y Recuperador de Datos

**Sortify** es una herramienta potente y ligera diseñada para poner orden en el caos de archivos y rescatar información perdida. Combina una lógica avanzada de organización con la robustez del motor PhotoRec.

## ✨ ¿Qué hace a Sortify diferente?

* **Organización Avanzada:** Clasifica archivos por categorías (Imágenes, Código, ISOs, etc.) moviéndolos a carpetas específicas de usuario o dentro del mismo directorio.
* **Detección de Duplicados (MD5):** No se deja engañar por nombres iguales. Calcula el hash MD5 de cada archivo para eliminar duplicados reales y ahorrar espacio.
* **Limpiador de Nombres:** Detecta y elimina automáticamente sufijos como " - copia", "(1)", o "kopie" para que tus archivos tengan nombres limpios.
* **Recuperación "Invisible":** Gestiona la descarga, instalación y ejecución de PhotoRec de forma automática. Tú solo eliges el disco y Sortify se encarga del resto.
* **Modo Seguro:** Incluye un botón de **Deshacer** que revierte la organización de archivos si cometiste un error.

## 🛠️ Requisitos Técnicos
* **SO:** Windows (usa librerías nativas como `ctypes.windll`).
* **Lenguaje:** Python 3.x.
* **Librerías:** Necesitas instalar `customtkinter` para la interfaz gráfica.

## 🚀 Instalación rápida
1. Clona el repositorio.
2. Instala la interfaz: `pip install customtkinter`.
3. Ejecuta: `python código_base.py`.
